                             Retail Sales Analysis Dashboard

 Overview  :- This project analyzes retail sales data using Python, MySQL, and Power BI.

## Tools Used
- Python
- Pandas
- MySQL
- Power BI

## Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- SQL Analysis
- Interactive Dashboard

## Dashboard Includes
- Top 10 Products by Revenue
- Sales by Country
- Top 10 Customers
- Monthly Sales Analysis
- Quantity vs Revenue Analysis
